Docs API Bambu Ecommerce
Auth

api/auth/register (POST)

Registrar nuevo usuario

Parametros:

  • name (string), requerido
  • lastname (string)
  • email (string), requerido
  • password (string), requerido, minimo 8 caracteres

api/auth/login (POST)

Iniciar sesion

Parametros:

  • email (string) requerido
  • password (string) requerido, minimo 8 caracteres

api/auth/renew (POST)

Renovar token de autorizacion

Sin parametros

api/auth/recover-password (POST)

Pedido de recuperación de contraseña via email

Parametros:

  • email (string) required

api/auth/resetPassword (GET)

Validacion de token enviado por email

Parametros:

  • token (string)

api/auth/resetPassword (POST)

Cambiar contraseña

Parametros:

  • token (string) required
  • password (string) required
Products

/api/products (POST)

Agregar producto

Parametros:

  • title (string) requerido
  • description (string) requerido
  • code (number) requerido
  • price (number) requerido
  • stock (number) requerido
  • category (string) requerido
  • thumbanail (string)
  • owner (_id) requerido: _ID de user valido para MongoDB

/api/products (GET)

Obtener todos los prodcutos de la base de datos con paginacion

Sin parametros

/api/products/:PID (POST)

Obtener un producto especifico por _ID

Parametros:

  • pid (string) requerido: _ID de product valido para MongoDB

/api/products/:PID (PUT)

Modificar la informacion de un producto en la base de datos

Parametros:

  • pid (string) requerido: _ID de product valido para MongoDB
  • title (string)
  • description (string)
  • code (number)
  • price (number)
  • stock (number)
  • category (string)
  • thumbanail (string)

/api/products/:PID (DELETE)

Eliminar un producto de la base de datos

Parametros:

  • pid (string) requerido: _ID de product valido para MongoDB
Carts

/api/carts/:CID (POST)

Obtener un carrito por _ID

Parametros:

  • pid (string) requerido: _ID de cart valido para MongoDB

/api/carts/:CID/product/:PID (POST)

Agregar un producto al carrito

Parametros:

  • cid (string) requerido: _ID de cart valido para MongoDB
  • pid (string) requerido: _ID de product valido para MongoDB

/api/carts/:CID/product/:PID (DELETE)

Eliminar un producto del carrito

Parametros:

  • cid (string) requerido: _ID de cart valido para MongoDB
  • pid (string) requerido: _ID de product valido para MongoDB

/api/carts/:CID/product/:PID (PUT)

Modificar la cantidad de un producto del carrito

Parametros:

  • cid (string) requerido: _ID de cart valido para MongoDB
  • pid (string) requerido: _ID de product valido para MongoDB
  • quantity (number) requerido: cantidad del producto

/api/carts/:CID/product/:PID (DELETE)

Realizar la compra del carrito

Parametros:

  • cid (string) requerido: _ID de cart valido para MongoDB

/api/carts/:CID (DELETE)

Vaciar el carrito

Parametros:

  • cid (string) requerido: _ID de cart valido para MongoDB
